# FKC! Fighting Krazy Chickens! 🐔

Landing page for **FKC! Fighting Krazy Chickens!** — a comedic bullet-heaven action game by [Sunblind Games](https://sunblindgames.github.io).

## About

FKC! is a physics-based action game where enemies are your ammunition. Grab chickens, throw them, and cause chaos.

- 🎮 **Adventure Mode** — Solo campaign with waves, bosses, and upgrades
- 🤝 **Co-op Mode** — Team up with friends to survive
- 👑 **King of the Ring** — PvP arena madness
